# Real-Time Chat Application (Socket.io)

A simple real-time chat application using **HTML, CSS, JavaScript, Node.js, Express, and Socket.io**. Users can join rooms by entering a **username** and **room ID**, then chat in real-time.

## Features
- **Real-time messaging** powered by Socket.io.
- **Room-based chat** where users can join specific rooms.
- **Enter key binding** for connecting and sending messages (no buttons for these actions).
- **Smooth scrolling chatbox** without visible scrollbars.
- **Disconnect button** to leave the chat.
- **Stop server button** to terminate the server.
